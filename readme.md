# 🏦 Système de Scoring Crédit - "Prêt à dépenser"

![Python](https://img.shields.io/badge/python-3.10-blue.svg) ![Flask](https://img.shields.io/badge/flask-2.0-black.svg) ![Streamlit](https://img.shields.io/badge/streamlit-1.20-red.svg)

Ce projet implémente un outil de **Scoring Crédit** pour prédire la probabilité de faillite d'un client de manière automatique.

## 🚀 Architecture du Projet
* **Backend :** API Flask déployée sur **Render**.
* **Frontend :** Dashboard interactif avec **Streamlit**.
* **Modèle :** LightGBM optimisé avec un score métier (coût des Faux Négatifs).
